# 开发日记

我是云中江树。TokenCode 不只想开源代码——代码只留下结果，留不下**当时为什么这么决定**，而那些取舍恰恰是最值钱的部分。所以我每天把三件事原样记下来：

1. **做了什么**：具体到包、到机制、到行为。
2. **状态**：只写真话——测过的说怎么测的，没测的就说没测，不把没验证的说成完成。
3. **为什么**：选了什么、放弃了什么、付了什么代价。

配合 [`../STATUS.md`](../STATUS.md)（此刻的能力快照）和 [`../ROADMAP.md`](../ROADMAP.md)（整张路线图），你应该能在半小时内接上手。

格式很简单：一天一篇，文件名 `YYYY-MM-DD.md`，文末给想接力的人留切入点和坑。
